📖 使い方・仕様

🖼 キャラクターの登録

キャラの登録方法は2種類あります。
① 画像から登録：「ここをタップして画像を選択」で画像を選ぶと登録モーダルが開きます。キャラ名・タイプ・ランク・性別（任意）・凸最大値（任意）を設定して「登録」を押すと一覧に追加されます。複数枚選択した場合は1枚ずつ順番に登録できます。
② 手動登録（画像なし）：画像がなくてもキャラ名・タイプ・ランク・性別・凸を入力して登録できます。抽出時はグレーのアイコンで表示されます。
ファイル名自動認識：キャラ名_タイプ_ランク_性別_凸.jpg の形式で命名されたファイルは、各項目が自動入力されます（ランク以降は省略可）。
　ランク：A / 4 / 星4 / ☆4 / ★4　または　S / 5 / 星5 / ☆5 / ★5
　性別：女 / ♀ / メス / 雌 / 女性 / female　または　男 / ♂ / オス / 雄 / 男性 / male
　凸：0〜6の数字　または　無凸 / 1凸 / 2凸 / 3凸 / 4凸 / 5凸 / 完凸
データの保存：登録データはブラウザに自動保存されます。次回以降は開くだけでそのまま使えます（※ブラウザのキャッシュをクリアすると消えます）。

✏️ キャラクターの編集・削除

キャラ一覧の画像（またはアイコン）をクリック／タップすると編集モーダルが開き、以下の項目を変更できます。
　・ キャラ名　・ タイプ　・ ランク（A / S）　・ 性別（♂ / ♀）　・ 凸最大値（無凸〜完凸）
各キャラ右下の「ON / OFF」トグルをオフにすると、そのキャラは一時的に抽出対象から外れます。「🗑 削除」で完全に削除できます。

⭐ キャラクター設定（ランクフィルタ）

抽出対象をランクで絞り込めます。
　・ 全画像：登録済み全キャラが対象
　・ Aのみ：ランクAのキャラのみが対象
　・ Sのみ：ランクSのキャラのみが対象
ランク未設定のキャラは「全画像」時のみ対象になります。

📝 弧盤候補

抽出時にキャラとペアで表示されるテキスト（弧盤）の候補一覧です。
　・ チェックボックスで使用する候補を個別にオン／オフできます
　・ 右上の「弧盤候補」トグルをオフにすると弧盤機能ごと無効化され、キャラのみ抽出されます
　・ 候補の追加・削除も自由にできます

⚙️ 抽出設定（モード）

通常抽出：タイプが一致するキャラと弧盤をランダムに1〜4組抽出します。
前半＋後半同時抽出：前半・後半それぞれにキャラと弧盤を抽出します。編成を前後半で分けたいときに使います。
4人編成：キャラを4人固定で抽出します。「前半後半に対応する」をオンにすると前半4人・後半4人の計8人が一度に抽出されます。マルチプレイ表示では各プレイヤー分の4人編成をまとめて抽出でき、2人用（1P・2P 各4体 計8体）・3人用（1P〜3P 各4体 計12体）・4人用（1P〜4P 各4体 計16体）から選択可能です。各プレイヤーの結果はカラー枠で区別して表示されます。
単騎抽出：全キャラの中から1体だけ抽出します。弧盤候補がオンの場合、タイプ一致の弧盤も同時に抽出されます。連続で押すと前回と同じキャラが連続して出ないよう除外機能と連動します。

🎯 凸ランダム機能

抽出時にキャラの凸数をランダムで表示できます。「凸ランダム機能を使用する」をオンにして使います。
完全ランダム：各キャラの凸最大値の範囲内で無凸〜上限までランダム抽出します。
範囲指定ランダム：チェックボックスで抽出したい凸数を指定できます。キャラの凸最大値を超える値は自動的に除外されます。
キャラ別事前設定：登録・編集時に設定した凸最大値をもとに、キャラごとの上限内でランダム抽出します。キャラ画像をクリックして編集モーダルから設定してください。
※ どのモードでも、キャラに凸最大値が設定されている場合はその上限を超えることはありません。

🚫 除外機能

「一度抽出した項目を次の1回だけ除外する」をオンにすると、抽出されたキャラは次の1回は抽出対象から外れます。連続で同じキャラが出るのを防ぎたいときに便利です。
　・ 全抽出モード（通常・前半後半・4人・単騎）に対応しています
　・ 「除外リセット」ボタンで除外状態をいつでも手動で解除できます
　・ オフにすると全モードで完全ランダム（被りあり）になります

🔒 追加縛り機能

抽出結果に一緒に表示するプレイ縛りを自由に登録できます。メモ欄に1行ずつ縛りを入力し、「追加縛りを使用する」をオンにすると抽出時に縛りが表示されます。
抽出モードは3種類：
　・ ランダムで1つ：登録された縛りの中からランダムで1つ表示
　・ 指定した数：表示する縛りの数を手動で指定
　・ 無制限（数もランダム）：数も内容もランダムで表示
特殊キーワード（性別縛り）：
　メモに 男性のみ または 女性のみ と書くと、そのキーワードが選ばれたとき抽出プールをその性別のキャラに絞り込みます。性別はキャラの登録・編集時に「性別」欄（♂/♀）で設定してください（未設定キャラはフィルタ対象外）。

✏️ キャラクター編集

無凸〜設定した凸数の範囲でランダム抽出。
